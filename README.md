# HTB Writeups

Documentación de mis prácticas en [Hack The Box](https://www.hackthebox.com/) — writeups organizados por laboratorio y nivel, más apuntes teóricos por temática.

## Estructura

```
htb-writeups/
├── labs/
│   └── starting-point/
│       ├── very-easy/
│       │   ├── Meow.md
│       │   └── Fawn.md
│       ├── easy/
│       ├── medium/
│       └── hard/
└── teoria/
    ├── redes/
    ├── reconocimiento/
    ├── explotacion/
    ├── post-explotacion/
    └── herramientas/
```

`labs/` contiene los writeups de las máquinas completadas, organizados por sección de HTB y nivel de dificultad. `teoria/` contendrá apuntes sobre los módulos y contenido teórico de HTB Academy y Starting Point, organizados por temática.

## Entorno

- Conexión a laboratorios HTB via OpenVPN: `sudo openvpn --config archivo.ovpn`
- Sistema: Ubuntu Noble 24.04 ARM64
